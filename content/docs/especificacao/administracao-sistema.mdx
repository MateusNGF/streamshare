# Administração da Plataforma

Algumas funcionalidades do StreamShare são gerenciadas centralmente pela equipe da plataforma e não estão disponíveis para utilizadores finais. Esta seção descreve o que é gerenciado a nível de plataforma e como isso afeta os Organizadores.

---

## Catálogo Global de Streamings

O Catálogo Global é o repositório central que define quais as plataformas de streaming que os Organizadores podem gerenciar. Ele é mantido e atualizado regularmente pela equipe do StreamShare.

### O que está no Catálogo

O catálogo inclui as principais plataformas de streaming disponíveis no mercado, como Netflix, Spotify, Disney+, entre muitas outras. Para cada plataforma, o catálogo regista:

- **Nome oficial** da plataforma;
- **Logotipo** para identificação visual;
- **Categoria** (Vídeo, Música, Jogos, etc.);
- **Indicação de conteúdo adulto**, quando aplicável, para fins de filtragem.

### Impacto para os Organizadores

Ao adicionar um novo serviço, o Organizador seleciona a plataforma base no catálogo e herda automaticamente as suas informações. Não é possível adicionar um serviço que não esteja no catálogo.

> Se um serviço que usa estiver em falta no catálogo, pode solicitá-lo à equipa de suporte do StreamShare. Novos serviços são adicionados regularmente.

### Histórico e Preservação de Dados

Se uma plataforma for descontinuada ou removida do catálogo, os registros históricos das contas que utilizavam esse serviço são preservados. Relatórios e faturas antigas continuam a exibir os dados corretos.
